# 🌿 EcoTech - Educação Tecnológica & Ambiental

O **EcoTech** é uma plataforma educativa interativa voltada para o ensino de tecnologia com foco em sustentabilidade ambiental. Desenvolvido por estudantes de engenharia, o projeto busca demonstrar como o domínio de ferramentas digitais e práticas conscientes de TI podem gerar um impacto positivo direto na preservação do planeta.

---

## 🚀 Funcionalidades Principais

- **Calculadora de Impacto Ambiental**: Calcule a quantidade de folhas de papel economizadas e árvores salvas ao ano substituindo processos físicos por planilhas digitais (Excel).
- **Conteúdo Educativo Integrado**: Cursos focados na conscientização e no desenvolvimento de habilidades tecnológicas essenciais:
  - **Excel para Gestão Verde**: Automação e digitalização de processos para redução do uso de papel.
  - **Tecnologia Básica e E-waste**: Prevenção do descarte precoce de componentes eletrônicos.
  - **Computação em Nuvem (Cloud)**: Redução de infraestrutura local e eficiência energética.
- **Interface Responsiva e Dinâmica**: Suporte a temas claro/escuro e efeitos visuais modernos (glassmorphism).

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica das páginas.
- **CSS3 (Vanilla)**: Design premium responsivo, estilização customizada com efeitos glassmorphism e controle de temas.
- **JavaScript (Vanilla)**: Lógica interativa, calculadora de impacto ambiental e alternância de temas (Claro/Escuro).
- **Google Fonts (Inter & Montserrat)**: Tipografia moderna e limpa.

---

## 💻 Como Executar o Projeto

Como o projeto é desenvolvido utilizando tecnologias web puras (HTML/CSS/JS), nenhuma instalação complexa ou servidor backend é estritamente necessário para a visualização local:

1. Clone ou faça o download deste repositório.
2. Navegue até a pasta raiz `EcoTechFinal`.
3. Dê um duplo clique no arquivo `homePage.html` para abri-lo diretamente no seu navegador de preferência, ou utilize uma extensão como a **Live Server** no VS Code para uma melhor experiência de desenvolvimento.

---

## 💡 Sobre o Projeto

Este projeto foi construído sob a premissa de que a tecnologia não deve apenas evoluir, mas também atuar como aliada da sustentabilidade. Ao educar usuários sobre eficiência energética no uso de computadores, computação em nuvem e digitalização de processos, o EcoTech ajuda a moldar a próxima geração de profissionais de tecnologia ambientalmente conscientes.
